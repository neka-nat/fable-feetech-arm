# 6-Axis Robot Arm (Feetech STS3215 × 6)

`STS3215_03a.step`(Feetech STS3215 サーボ)を6個使った6軸ロボットアームのCADモデルです。
ブラケットの取り付けインターフェースはすべて STEP モデルから実測した寸法に合わせてあります。

## 生成方法

```bash
uv run python robot_arm.py                            # 直立姿勢で出力
uv run python robot_arm.py --pose "0,-40,70,30,50,0"  # J1..J6 [deg] を指定
```

出力:
- `robot_arm.step` — 指定ポーズの全体アセンブリ(サーボ6 + ブラケット7、部品ラベル・色付き)
- `out/parts/*.step` — ブラケット単品(3Dプリント用、ポーズ非依存)
- 実行時にペアワイズ干渉チェックを行い、重なりがあれば報告します(`--no-check` でスキップ)

## 3Dビューアでジョイントを動かす

ブラウザ上のスライダーで6軸をリアルタイムに動かせます(VS Code不要):

```bash
uv run python arm_viewer_viser.py
```

http://127.0.0.1:8080 をブラウザで開くと、右上のJ1〜J6スライダーで関節を
動かせます。メッシュは起動時に一度だけテッセレートし(初回は ~1分)、以後は
関節フレームの回転のみ更新するので操作はリアルタイムに反映されます。
`reset pose` ボタンで全関節を0(直立姿勢)に戻せます。

干渉チェックは行わないので、当たりが疑わしいポーズは `robot_arm.py --pose` で
確認してください(ペアワイズ干渉チェック付き)。

Pythonからは `from robot_arm import build_arm` で任意ポーズの
アセンブリ(`Compound`)を取得できます。

## 軸構成(ゼロ姿勢 = 直立)

| 軸 | 種別 | 軸方向 | 駆動部品 |
|----|------|--------|----------|
| J1 | ベースヨー | Z | `base` → `b1_shoulder_mount` |
| J2 | ショルダーピッチ | Y | `b1` → `b2_upper_arm`(リンク長 110 mm) |
| J3 | エルボーピッチ | Y | `b2` → `b3_forearm` |
| J4 | リストロール | Z | `b3` → `b4_wrist_mount` |
| J5 | リストピッチ | Y | `b4` → `b5_wrist_link` |
| J6 | ツールロール | Z | `b5` → `b6_tool_flange` |

全高(直立時)約 371 mm、ショルダーからフランジまでの実効リーチ約 277 mm。

## サーボとの取り合い(STEPから実測)

- **ドライブホイール**: Ø20、面 z=+18.7、Ø2.5 の4穴(9.9 mm 角配置)→ **M3 ねじ**(穴はM3タップ下径)。
  中心ねじ頭(z=+20.2 まで突出)用に Ø9.5 の逃がし座繰りを各ブラケットに設けています。
- **アイドラホイール**: Ø20、面 z=−17.7、同一穴パターン。中心ボス Ø6(z=−18.3)用の Ø7 逃がし付き。
  U字ブラケットはホイール/アイドラ両面を掴む両持ち支持です。
- **上面タブ**: 面 z=+15.9、Ø1.5 下穴 ×4(−29.0/−8.3, ±10.25)→ **M2 タッピングねじ**(皿、面一)。
- **下面タブ**: 面 z=−15.9、Ø1.5 下穴 ×4(−32.8/−8.3, ±10.25、上面と非対称)。
- **干渉回避**: ケース上面リブ(z=+17.0 まで)・コネクタ部(z=−19.4 まで)にはクランプ板に逃がし窓をカット。
  コネクタ窓はケーブル引き出し口を兼ねます。

## ねじBOM(目安)

| 用途 | ねじ | 数量 |
|------|------|------|
| ホイール/アイドラ固定 | M3×6 キャップ/ボタン | 4×10箇所 = 40 |
| サーボタブ固定 | M2×8〜10 皿タッピング | 4×10箇所 = 40 |
| ベース固定 | M4×12 + 座繰り | 4 |

## 組立・運用ノート

- ホイール用 M3 はスラブ/フランジに設けた Ø6 のドライバーアクセス穴から締めます。
  デッキ上のサーボ(J4・J6)はタブねじを締めてから次のブラケットを取り付けてください。
- 可動範囲の目安: J1/J4/J6 はほぼ全周(配線次第)、J2/J3/J5 は ±100° 程度で
  ブラケットウェブがサーボ尾部・前段リンクに当たります。制御側でリミット設定してください。
- ブラケット推奨: PETG/PLA、壁4周・インフィル40%以上。プレート厚は基本4 mm、
  タブクランプ板はホイール面との段差制約(2.8 mm)から 2.5 mm。
